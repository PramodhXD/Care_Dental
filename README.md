# Care Multi Speciality Dental Hospital Website

Official website for Care Multi Speciality Dental Hospital, Tenali. Built with Next.js App Router and Tailwind CSS.

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- Swiper

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Project Structure

- `app/`: App Router pages and global styles
- `components/layout/`: Header and footer
- `components/sections/`: Home page sections
- `components/common/`: Shared UI elements
- `data/`: Static content data used by sections
- `public/images/`: Clinic and brand assets
